# Proactive remediation: Collect low disk space inventory

Collect Windows disk usage, fails if system disk space usage is above defined disk space usage threshold.

## Script package information

Disk Space Usage Threshold in percentages (0-100): **80%**

External logging: **No**

## Script package properties

### Basic

Name: **Collect low disk space inventory**

Description: **Collect Windows disk usage, fails if system disk space usage is above 80%.**

Publisher: **Jesper Nielsen**

### Settings

Detection script: **Yes**

Remediation script: **No**

Run this script using the logged-on credentials: **No**

Enforce script signature check: **No**

Run script in 64-bit PowerShell: **No**

### Assignments

Schedule: **Weekly**
